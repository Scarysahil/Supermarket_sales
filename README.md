# Supermarket_sales
Supermarket Sales Analysis using Python &amp; Data Analytics A data analysis project that explores supermarket sales data to identify trends in revenue, customer behavior, product performance, and branch-wise sales using Python (Pandas, Matplotlib, Seaborn).

# 🧾 Supermarket Sales Analysis

## 📊 Project Overview
This project focuses on analyzing supermarket sales data to extract meaningful business insights. The goal is to understand customer behavior, sales trends, product performance, and branch-wise performance using data analysis techniques.

---

## 🎯 Objectives
- Analyze total sales and revenue trends
- Identify best-selling product lines
- Understand customer purchase behavior
- Compare performance across branches and cities
- Visualize key business insights using graphs

---

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset Information
The dataset includes supermarket transaction records such as:
- Invoice ID
- Branch & City
- Customer type
- Gender
- Product line
- Unit price
- Quantity
- Total sales
- Date & Time
- Payment method

---

## 📈 Key Insights
- Which product line generates the highest revenue
- Peak sales hours and days
- Customer segmentation (Member vs Normal)
- Branch-wise performance comparison
- Payment method distribution

---

## 📊 Visualizations
The project includes:
- Bar charts (sales by product line)
- Pie charts (payment methods)
- Line charts (sales trends)
- Heatmaps (correlation analysis)
